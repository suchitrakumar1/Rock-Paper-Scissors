# 🪨 Rock Paper Scissors (JavaScript Project)

A simple Rock Paper Scissors game built using **HTML, CSS, and JavaScript**.  
This project focuses on understanding **DOM manipulation, event handling, and basic game logic** without relying on tutorials.

---

## 🚀 Features

- Click-based user interaction (Rock, Paper, Scissors)
- Random computer choice generation
- Real-time result display (Win / Lose / Draw)
- Score tracking for both user and computer
- Dynamic UI updates using JavaScript

---

## 🧠 What This Project Demonstrates

- Selecting elements using `querySelector`
- Handling user events with `addEventListener`
- Using conditional logic for game outcomes
- Generating random values using `Math.random()`
- Updating UI dynamically using `.innerText` and `.style`

---

## 🎮 How It Works

1. User clicks on Rock, Paper, or Scissors
2. Computer randomly selects a choice
3. Game compares both inputs:
   - Rock beats Scissors
   - Paper beats Rock
   - Scissors beats Paper
4. Result is displayed and score updates accordingly
